# Space Airport

This project is built with React, React Hooks, Redux, Redux-thunk, Redux Toolkit, Chakra UI
- React, React Hooks -> frontend library
- Redux, redux-thunk, Redux Toolkit -> global state management
- Chakra UI -> UI library and RWD design
